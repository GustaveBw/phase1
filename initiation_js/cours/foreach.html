FOREACH

La boucle FOREACH est une boucle utilisée pour les tableaux dans différents langages de programmation. En JS, cette boucle est liée aux objets de type array

Syntaxe: 

tableau.foreach( function(valeur, indice...) {
        instructions...
        })

Dans FOREACH, on va déclarer une fonction qui va s'appliquer sur toutes les valeurs du tableau.

FOR OF

une autre version de la boucle FOREACH en JS est la boucle FOR OF

Syntaxe: 

            for(const valeur of tableau){
                ...instructions...
            }

Pour chaque valeur du tableau, on va éxecuter les instructions de la boucle.

ici on ne récupère pas les indices, uniquemebt les valeurs qui sont affectées a une nouvelle variable ( dans l'exemple, c'est la variable jour).
