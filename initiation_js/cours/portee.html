Portée des variables (scope)

  • Une variable décclarée dans une fonction n'est accesible que dans cette fonction. On dit que c'est une variable locale a la fonction.
  • Une variable declarée avec le mot clé var est une véritable globale.Elle est accessible partout.
  • une variable déclarée avec le mot-clélet n'est accessible que dans le block ou elle a été déclarée. Un block est Délimité par des {} (accolades, curly brackets en anglais.) On ne peut pas redéclarer une variable déclarée avec let dans le meme bloc.
  • Le mot-clé const permet de déclarer des constantes. Une constante est une variable dont la valeur ne peut pas etre modifié
